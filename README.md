# 🧠 Comportamental AL

## Análise Comportamental Profissional

Aplicativo PWA (Progressive Web App) para análise de linguagem corporal e comportamento, baseado em estudos científicos de Paul Ekman, Joe Navarro, Allan Pease e Desmond Morris.

---

## 📱 Como Instalar no Celular

### Android (Chrome):
1. Abra o app no navegador Chrome
2. Toque no menu (⋮) no canto superior direito
3. Selecione **"Adicionar à tela inicial"** ou **"Instalar app"**
4. O ícone aparecerá como um app nativo

### iPhone (Safari):
1. Abra o app no Safari
2. Toque no botão **Compartilhar** (□⬆️)
3. Role para baixo e toque em **"Adicionar à Tela de Início"**
4. Toque em **"Adicionar"** no canto superior direito

### Desktop (Chrome/Edge):
1. Abra o app no navegador
2. Clique no ícone de instalação (📥) na barra de endereço
3. Ou vá em Menu → "Instalar Comportamental AL"

## 🎯 Funcionalidades

- 📸 **Análise de Fotos** - Capture uma pose ou expressão estática
- 🎥 **Análise de Vídeos** - Grave movimentos, gestos e padrões temporais
- 🖼️ **Galeria** - Selecione fotos ou vídeos já existentes
- 📊 **Relatórios Profissionais** - Scores por categoria com interpretação
- 📚 **Guia de Referência** - Banco de sinais corporais com fundamentação científica
- 📋 **Histórico** - Salve e consulte análises anteriores
- 🔒 **100% Privado** - Tudo processado localmente no dispositivo
- 📴 **Funciona Offline** - PWA instalável, não precisa de internet

---

## 🧬 Baseado em Estudos de:

- **Paul Ekman** - Sistema FACS, Emoções Reveladas, microexpressões
- **Joe Navarro** - O Que Todo Corpo Diz, gestos de acalmação
- **Allan Pease** - Linguagem Corporal no Trabalho, gestos ilustradores
- **Desmond Morris** - O Macaco Nu, gestos universais
- **Edward Hall** - Proxêmica, uso do espaço pessoal
- **Ray Birdwhistell** - Kinesia, análise do movimento corporal

## ⚠️ Aviso Ético

Este app é uma ferramenta educacional e profissional. A análise de linguagem corporal:

- Não é uma ciência exata
- Deve considerar sempre o contexto cultural e individual
- Não substitui avaliação de especialistas qualificados
- Não deve ser usada para discriminar ou julgar unilateralmente
- Sinais individuais não determinam intenções — observe padrões múltiplos

- HTML5, CSS3, JavaScript (Vanilla)
- Progressive Web App (PWA)
